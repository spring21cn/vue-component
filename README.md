# vue-component
这是一套基于 Vue.js 的高质量 开源UI组件库。

 
 
#安装
在页面上引入 js 和 css 文件即可开始使用。

~~~
<!-- import css -->
<link rel="stylesheet" href="./css/component.min.css">
<!-- import Vue -->
<script src="./js/vue-all.min.js"></script>
<!-- import component -->
<script src="./js/vue-component-all.min.js"></script>
~~~
